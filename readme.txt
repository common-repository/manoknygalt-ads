Šaunus pluginas rodantis ManoKnygos.lt reklamą jūsų bloge ir leidžiantis uždirbti manoknyga.lt litų iš kiekvieno jūsų atvesto kliento.

== Installation ==

1. Parsisiunčiame įskiepį
2. Įkeliam išarchyvuotą aplanką į wp-content/plugins/
3. Įjungiam įskiepį Plugins skiltyje (bloge)
4. Nustatome reklamos parametrus Nuostatos -> ManoKnyga.lt
5. Įkeliame reklamą į blogą per Išvaizda -> Valdikliai (valdiklio pavadinimas - ManoKnyga.lt)
